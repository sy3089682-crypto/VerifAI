## Hacker News Post

Title: I built VerifAI entirely from my phone — the trust layer for AI code

Body:

41% of committed code is now AI-generated. Only 33% of developers trust it.

AI tools generate code faster than we can review it. Bugs that look correct, pass all tests, and cause production incidents. Black box decisions hidden inside confident-sounding output.

I built VerifAI to solve this — and I did it entirely from my Android phone (Termux + opencode). No laptop, no desk, no funding.

What it does:
- Records every AI decision with full context (model, prompt, output, drift analysis)
- Provides a queryable audit trail filtered by model, tag, or time range
- Enforces custom invariants on AI-generated code (security, style, correctness)
- Automatically detects drift between prompt intent and output

It's free, open source (MIT), and runs entirely locally. No servers, no API keys, no signup.

Sometimes the tool the industry needs most gets built by someone with the least resources — because they're too busy solving the problem to worry about what's "supposed to" be possible.
